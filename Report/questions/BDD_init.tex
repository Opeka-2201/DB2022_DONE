notre compte sur ces machines :

  $ ssh s191230@ms801.montefiore.ulg.ac.be
  $ cd WWW
  $ mysql -h ms8db -u group25 -p
  Enter password: 3uCTA8L2ID
  mysql> source ./script.sql
\end{verbatim}
